# S&P 500 ML Project

## Overview
I started this project by analyzing S&P 500 data using basic Python and simple statistics. As I learned more concepts on the Coursera , I expanded it into a machine learning project to predict whether the market will go up or down the next day.

## What I Did
- Collected data using yfinance  
- Explored trends with basic Python and pandas  
- Added features like averages and volatility  
- Built a Random Forest machine learning model  
- Evaluated model performance  

## Tools
- Python  
- Pandas  
- Scikit-learn  
- Matplotlib  

## Conclusion
The model achieves around 50–60% accuracy, showing how challenging it is to predict the stock market.
